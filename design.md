# VG CS Classes Website Design

## Design Philosophy
- **Color Scheme**: Professional education theme with blue (#1e3a8a) as primary and orange (#f97316) as accent
- **Typography**: Modern sans-serif fonts (Inter, system fonts) for clean readability
- **Visual Language**: Clean, professional, and inspiring educational atmosphere
- **Animations**: Smooth scroll-triggered animations using Anime.js and CSS transitions

## Visual Effects
- **Hero Section**: Animated gradient background with floating particles
- **Text Effects**: Color cycling emphasis, split-by-letter stagger animations
- **Image Effects**: Ken Burns pan/zoom on hero images, infinite image scrollers
- **Interactive Elements**: Hover animations, floating call/Whatsapp buttons
- **Navigation**: Smooth transitions, mobile-friendly hamburger menu
